How to Play

1 / 5

Coloured Cages

The grid is divided into coloured groups called cages. Each cage of size N must contain the numbers 1 through N — no repeats within a cage.

No Touching

Two cells with the same number cannot touch — not even diagonally. This applies across all cages on the whole board.

Picking a Number

Tap a cell to select it. The number buttons light up to show only the numbers valid for that cage's size. Tap a number to place it.

Use Elimination

If a cage of 3 already has 1 and 2, the last cell must be 3. Start with the smallest cages — single-cell cages are always 1!

Wrong Answers

When the board is full, wrong cells turn pink. Fix them and the puzzle solves itself — no submit button needed.
